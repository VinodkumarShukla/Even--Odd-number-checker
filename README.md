# Even-Odd-number-checker

This Python program checks whether a given number is even or odd. Follow the prompts to input a number, and the program will determine and display whether the number is even or odd.

How to Use

Run the Program:
Execute the program in your preferred Python environment.
Enter a Number:
You will be prompted to input a number.
View the Result:
The program will display whether the entered number is even or odd.
Notes

The program uses the modulo operator (%) to check if the number is divisible by 2.
Ensure that you have a Python environment installed to run the program.
